1 - Création d'un client :

Créer un un client -> /createclient.html
Voir tous les clients - > /clients
Voir un seul client -> /client/:id
Delete un client -> /client/:id/remove
Update un client -> /client/:id/update
Delete plusieurs clients -> /clientdeleteMany (Utilisable uniquement sur reslet en post)
Update plusieurs clients -> /clientupdateMany (Utilisable uniqument sur reslet en put)
